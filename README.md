实际退化图像的恢复实验

1.很容易判断出图像的退化类型为运动模糊，因此选择维纳滤波进行图像恢复。
2.代码第一部分为读取图像，并显示出当前图像。
3.代码第二部分为模糊距离计算，方法为先对模糊图像一阶微分，这里使用的是sobel算子，
然后进行自相关运算，在得到的自相关函数图像中，两相关负峰之间的距离为模糊距离。经过
观察计算，得到该图模糊距离为16。
4.代码第三部分为模糊角度计算，方法为对灰度图像进行二维傅里叶变换，并对傅里叶变换
值的动态范围，对压缩后的结果进行循环移位，使其低频成分居中，用canny算子对压缩居
中后的频谱图像进行边缘检测使其二值化，将二值化后的频谱图做从1°~180°的radon变换，
找出radon变换后的矩阵中的最大值，求出其对应的列数n，通过公式
 tan(θ) = tan(φ ? 90°) × M/N  = tan(n ? 90°) × M/N 求出运动模糊方向。
5.代码第四部分为维纳滤波，将之前得到的模糊参数代入，其中信噪比因为在不知道原图像和噪声
的前提下较难计算，通过不断得到。最终得到了恢复之后的图像。
